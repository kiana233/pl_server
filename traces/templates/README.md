# Trace Templates

This directory contains schema examples only.

## Rules

- Templates are not real client trace.
- Templates are not confirmed protocol evidence.
- Templates must not contain account names, passwords, tokens, real IP addresses, device identifiers, or personal data.
- Templates exist only to show the expected sanitized JSONL shape.

## Allowed Files

- `sanitized-trace-event.example.jsonl`

Do not place raw captures, private traces, unreviewed traces, `.pcap`, `.pcapng`, `.har`, databases, or binary captures in this directory.
