# PLAN-0001: Overall Roadmap

## Goal

Rebuild the service foundation in a controlled order:

1. Repository workflow
2. Reference analysis
3. Protocol facts
4. Protocol layer
5. Logging and replay
6. Session state machine
7. AC0 handshake
8. AC63 login and character list
9. Character selection and enter-map
10. AC06 movement
11. Resource version contract
12. Inventory, equipment, NPC, warp, battle

## Principle

Do not patch symptoms randomly. Each change must be tied to a task, a known fact or recorded assumption, and an acceptance check.
