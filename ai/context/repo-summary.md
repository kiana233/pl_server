# Repository Summary

This repository was bootstrapped as an AI-Codex service rebuild workspace.

## Structure

- `ai/context/`: current facts and status
- `ai/plans/`: roadmap and architecture plans
- `ai/tasks/`: small executable Codex tasks
- `ai/reports/`: Codex execution reports
- `docs/`: long-lived project documentation
- `references/`: reference-server notes and analysis
- `traces/`: sanitized protocol traces and minimal samples
- `scripts/`: local workflow scripts
- `src/`: future server source code
- `tests/`: future unit and replay tests

## Implementation State

No server implementation exists yet. This is intentional for Phase 0.
