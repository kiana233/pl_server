# TASK-0001: Bootstrap AI-Codex Workflow

## Background

This repository is an empty Gitee repository for rebuilding a stable Wonderland Online compatible server foundation.

The workflow is:

- ChatGPT creates plans and task files.
- Codex runs locally and implements tasks.
- Gitee stores code, tasks, plans, and reports.
- The project must be driven by protocol facts and replay tests.
